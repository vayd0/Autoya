# Autoya

Selfbot Discord pour automatiser la commande `$p`.

## Avertissement

L'utilisation de selfbots viole les conditions d'utilisation de Discords. Utilisation à vos propres riques.

## Installation

```bash
pnpm install
```

## Configuration

Lancez `launch.bat` et répondez aux questions.
